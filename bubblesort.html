Bubble Sort

The bubble sort algorithm sorts an array by repeatedly finding the maximum element from the unsorted part and putting it at the end. The algorithm maintains two subarrays in a given array.
• The subarray which already sorted.
• The remaining subarray was unsorted.
In every iteration of the bubble sort, the maximum element from the unsorted subarray is picked and moved to the sorted subarray. The time Complexity of this algorithm is O(N*N) in best case, average case and worst case.

Time Complexity Of The Algorithm : O(N*N)

Space Complexity Of The Algorithm is : O(1)

import java.util.*; class BubbleSort { static void bubbleSort(int arr[]) { int n = arr.length; for (int i = 0; i < n - 1; i++) for (int j = 0; j < n - i - 1; j++) if (arr[j] > arr[j + 1]) { // swap arr[j+1] and arr[j] int temp = arr[j]; arr[j] = arr[j + 1]; arr[j + 1] = temp; } } /* Prints the array */ static void printArray(int arr[]) { int n = arr.length; for (int i = 0; i < n; ++i) System.out.print(arr[i] + " "); System.out.println(); } // Driver method to test above public static void main(String args[]) { int arr[] = { 5, 1, 4, 2, 8 }; bubbleSort(arr); System.out.println("Sorted array"); printArray(arr); } }
